# Technologies Reference

Updated: 2026-03-10
Owner: @ivans

## Core Framework
- React Native (Expo) - Mobile app framework
- TypeScript - Type safety
- Expo - Build and deployment platform

## State Management
- Redux Toolkit - Predictable state container
- React Redux - Redux integration for React
- Redux Thunk - Async action middleware (included in Toolkit)

## Storage & Persistence
- AsyncStorage - Local key-value storage
- Expo SecureStore - Secure credential storage

## UI & Styling
- React Native Animated - Built-in animations
- Themed styles (custom system) - Theme-aware styling
- CSS-in-JS patterns for web view

## Testing
- Jest - Test runner and framework
- React Native Testing Library - Component testing
- @testing-library/react-native - Testing utilities

## Tooling
- ESLint - Code linting
- Prettier - Code formatting
- Babel - JavaScript transpiler
- Jest - Unit and component tests

## Deployment
- Expo EAS - Build and submit service
- Vercel - Web deployment (for Expo Web)

## Where to verify exact versions
- Dependencies: [package.json](../../package.json)
- Expo config: [app.json](../../app.json)
- TypeScript: [tsconfig.json](../../tsconfig.json)
